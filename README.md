crop-detector
=============

Utility that finds cropped image coordinates inside original image

Working progress...

Usage
-----

```bash
crop-detector THRESHOLD ORIGINAL_IMAGE CROPPED_IMAGE
```

Example output: ```137 260 201 181```
It's: ```x y width height``` of cropped image
